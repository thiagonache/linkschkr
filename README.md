# linkschker
Program to check all links (href) for a given website.

Help
```
Usage of linkschkr:
  -debug
        Run in debug mode
  -quite
        Outputs nothing but the final statistics
  -recursive
        Run recursively (default true)
  -site string
        URL to check links
```

Run
```
./linkschkr -site https://bitfieldconsulting.com
```
